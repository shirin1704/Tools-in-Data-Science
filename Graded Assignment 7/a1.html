Data Analysis Code


import pandas as pd

# Load financial data
data = {
    "quarter": ["Q1 2025", "Q2 2025", "Q3 2025", "Q4 2025"],
    "revenue": [5000000, 5750000, 6100000, 7000000]
}

df = pd.DataFrame(data)

# Calculate quarterly growth
df['growth'] = df['revenue'].pct_change() * 100
print(df[['quarter', 'growth']])
This Python snippet shows how we computed quarterly revenue growth using Pandas.
